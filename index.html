My Portfolio

Hello! I'm Jana Vande Velde, and I'm thrilled to welcome you to my portfolio. Here, you'll find a collection of projects I've been passionately working on during my journey as an applied computer science student at Thomas More.

Website MJ entertainment

I created a website for MJ Entertainment, offering kids' party entertainment like balloon twisting, circus shows, and magic tricks. The development process involved the use of Bootstrap, HTML, CSS, and JavaScript, enhancing the site's visual appeal and functionality.

React Native Ready2Cook

Ready2Cook is an app that helps users find cooking inspiration based on the ingredients they already have at home. I developed this app using React Native and integrated an API to fetch recipes.

PWA TravelStars

TravelStars is an app where users can post and share their travel blogs. During this project, I worked with new technologies such as caching, IndexedDB for local storage, and API integration.

PHP CompeteMore

For our PHP project, we developed a competition platform for Thomas More, where students can join a variety of challenges like photography contests, hackathons, and coding competitions. Through this project, I gained experience with Laravel and I also learned more about implementing CRUD functionalities.

Dino Game

I recreated the popular Chrome Dino game with modern graphics, keeping the development simple using only plain HTML, CSS, and JavaScript. Through this project, I learned more about CSS keyframes and animations, and also found some great sources for free game sprites.

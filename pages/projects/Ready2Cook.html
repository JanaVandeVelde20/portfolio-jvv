Ready2Cook


I'm developing a React Native app called Ready2Cook, designed to help users find recipes based on the ingredients they already have at home. By entering the ingredients you have, the app uses the Spoonacular API to search for recipes that make the most of those ingredients, minimizing food waste. While the app is still a work in progress, I will be adding more details about the project soon. Users will also be able to save their favorite recipes, and I plan to integrate filters for dietary preferences, such as allergies or vegetarian options.

Used technologies: I developed the Ready2Cook app using React Native for a smooth, cross-platform experience on both iOS and Android. I integrated the Spoonacular API to access a large recipe database, allowing the app to suggest recipes based on the ingredients users have. This setup ensures a responsive and user-friendly experience.

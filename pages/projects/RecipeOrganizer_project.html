Main purpose application

For DevOps & Security we got to choose our own project. I choose to design a recipe organizer using C# and Windows Forms.

In this recipe organizer, there are two kinds of recipes: standard and advanced. You can view, edit, create, or delete recipes of each type.

If you're curious about how the finished application looks, you can check it out in this video. Video Recipe Organizer App

Technical aspects

The application utilizes SQLite for local recipe storage, and I implemented the CRUD (Create, Read, Update, Delete) principles to manage recipe data. To facilitate database interactions, I employed the Object-Relational Mapping (ORM) tool Dapper. Additionally, I incorporated inheritance in the design, introducing a parent class named "Recipe" and a child class named "AdvancedRecipe." This inheritance structure allows for a hierarchical organization of recipe data within the application.


Reflection:
Through self-study, I gained valuable insights, in how to utilize databases with C# and Windows Forms. This skill is particularly important since the majority of modern applications rely on databases for their functionality.
